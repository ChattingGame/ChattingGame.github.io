# Chatting Game

Welcome to **Chatting Game**! This is a multiplayer chat application inspired by Discord, designed for users who want to connect, communicate, and have fun in a virtual environment. 

## Features

- **Text and Voice Chat**: Communicate with friends through text and voice channels.
- **User-Friendly Interface**: Easy-to-navigate layout that resembles popular chat applications.
- **Customizable Profiles**: Personalize your profile with avatars and status messages.
- **Public and Private Channels**: Join public chat rooms or create private channels for exclusive conversations.
- **Emoji Support**: Express yourself with a wide range of emojis.

## How to Play

To play the game, visit the [Chatting Game](https://ChattingGame.github.io) website. 

1. Ensure your browser supports WebGL.
2. Click "Play" to launch the game.
3. Join or create a chat room and start communicating!
